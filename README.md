Sistema de Cadastro XYZ Comércio 🚀
Este projeto é um sistema desktop de gerenciamento de clientes (CRUD) desenvolvido para o "XYZ Comércio". Ele possui uma interface gráfica amigável onde é possível cadastrar, visualizar, atualizar e deletar dados de clientes (CPF, Nome, E-mail, Telefone e Endereço). Todas as informações são salvas de forma segura em um banco de dados local.

🔌 Como fazer funcionar na sua máquina:
Instale o Python na sua máquina (certifique-se de marcar a opção "Add Python to PATH" durante a instalação, se estiver no Windows);

Faça o download ou clone este repositório para o seu computador;

Abra o terminal (ou prompt de comando) e navegue até a pasta onde o arquivo está salvo;

Execute o comando python nome_do_seu_arquivo.py (lembre-se de substituir pelo nome exato do seu arquivo Python);

Nota: As bibliotecas utilizadas (tkinter e sqlite3) já vêm embutidas por padrão na instalação do Python, então você não precisa rodar o pip install para elas. O banco de dados (xyz_comercio.db) será criado automaticamente na mesma pasta assim que você rodar o programa pela primeira vez.

📋 Pré-requisitos do sistema:
Sistema operacional Windows, macOS ou Linux.

Python instalado na máquina.

(Para usuários Linux): Em algumas distribuições, o Tkinter não vem pré-instalado com o Python. Se houver erro, basta rodar sudo apt-get install python3-tk.
